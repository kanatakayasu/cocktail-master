## 1. ウイスキー検定の概要

### 主催・運営

- **主催**: 一般社団法人ウイスキー検定実行委員会
- **運営**: ウイスキー文化研究所
- **監修**: 土屋守氏（ウイスキー評論家）
- **後援**: スコットランド国際開発庁

### 試験実施

- **実施**: 年2回（2月・9月）
- **形式**: 在宅試験（受験キット郵送）/ 一部会場試験

## 2. ウイスキーの基礎知識

### ウイスキーの定義

穀類を原料とし、**糖化→発酵→蒸留→樽熟成**を経て製造される蒸留酒。各国で法的定義が異なる。

### 原料による分類

| 分類 | 原料 | 蒸留方式 |
|---|---|---|
| **モルトウイスキー** | 大麦麦芽（モルト）100% | 単式蒸留器（ポットスチル） |
| **グレーンウイスキー** | トウモロコシ、小麦、ライ麦など | 連続式蒸留機 |

### 製品による分類

| 分類 | 内容 |
|---|---|
| **シングルモルト** | 単一蒸留所のモルトウイスキーのみ |
| **シングルグレーン** | 単一蒸留所のグレーンウイスキーのみ |
| **ブレンデッドモルト**（ヴァッテッドモルト） | 複数蒸留所のモルトウイスキーを混合 |
| **ブレンデッドグレーン** | 複数蒸留所のグレーンウイスキーを混合 |
| **ブレンデッドウイスキー** | モルト原酒＋グレーン原酒をブレンド |

### 世界5大ウイスキー

1. **スコッチウイスキー**（スコットランド）
2. **アイリッシュウイスキー**（アイルランド）
3. **アメリカンウイスキー**（アメリカ）
4. **カナディアンウイスキー**（カナダ）
5. **ジャパニーズウイスキー**（日本）

## 3. ウイスキーの製造工程

### (1) 製麦（モルティング / Malting）

- 大麦を**水に浸漬（スティーピング）** → 数時間ごとに水を抜いて空気にさらす
- **発芽室**で5日～1週間かけて攪拌・発芽促進
- 発芽で酵素（**アミラーゼ**等）が生成される
- **フロアモルティング**: 伝統的手法。床に大麦を広げ職人が手作業で攪拌
  - 現在もフロアモルティングを行う蒸留所: ボウモア、スプリングバンク、ラフロイグなど少数
- **キルン（乾燥塔）** で加熱して発芽停止
  - ピートを燃料にすると**ピート香**がつく
- 現在は専門の**モルトスター（製麦業者）** から仕入れるのが一般的

### (2) 粉砕（ミリング / Milling）

- 麦芽を**モルトミル**で粉砕し「**グリスト**」にする
- 理想的比率: **ハスク（殻皮）：グリッツ（粗粉）：フラワー（細粉）= 2：7：1**

### (3) 糖化（マッシング / Mashing）

- **マッシュタン（糖化槽）** で粉砕麦芽に温水を加えて攪拌
- 酵素がデンプンを糖に分解
- ろ過して**麦汁（ウォート / Wort）** を抽出
- 約20度まで冷却

### (4) 発酵（ファーメンテーション / Fermentation）

- **ウォッシュバック（発酵槽）** で麦汁に酵母を投入
  - 材質: 木製（オレゴンパイン、カラマツ等）またはステンレス
- 酵母が糖をアルコール＋二酸化炭素に分解
- 48～72時間でアルコール度数 **7～9%** の醪（もろみ）＝**ウォッシュ（Wash）** を生成

### (5) 蒸留（ディスティレーション / Distillation）

- **単式蒸留器（ポットスチル）** で2回蒸留（スコッチの場合）

**初溜（ウォッシュスチル）:**
- ウォッシュを加熱 → アルコール度数約 **20%** の**ローワイン**を生成

**再溜（スピリットスチル）:**
- ローワインをさらに蒸留 → **ミドルカット**で3段階に分離:

| 段階 | 名称 | 特徴 |
|---|---|---|
| 最初 | **フォアショット（ヘッド / 前留）** | 高アルコール、華やかだが不純物含む |
| 中間 | **ハート（ミドル / 中留）** | 製品となる本体部分 |
| 後半 | **フェインツ（テール / 後留）** | 重たい、オイリー |

- ハートのみを樽に詰める → アルコール度数 **65～70%** の**ニューポット（ニューメイクスピリッツ）**

### (6) 熟成（マチュレーション / Maturation）

- ニューポットに加水してアルコール度数 **63%前後** に調整
- オーク樽に詰めて貯蔵
- 樽成分が溶出 → 琥珀色、バニラ・カラメル等の風味付与

### (7) ブレンド・ヴァッティング

- **ブレンディング**: モルト原酒＋グレーン原酒を混合
- **ヴァッティング**: 同種の原酒同士を混合
- **マスターブレンダー**が数十～数百の原酒を組み合わせ

### (8) 後熟（マリッジ / Marriage）

- ブレンド後、再び樽に詰めて数か月～2年熟成
- 複数原酒を馴染ませ品質安定化

### (9) 加水・ろ過・瓶詰め

| 用語 | 説明 |
|---|---|
| **冷却ろ過（チルフィルタード）** | 低温にして澱や白濁成分を除去 |
| **ノンチルフィルタード** | ろ過を省略し、樽・発酵由来の風味成分を保持 |
| **カスクストレングス** | 加水せず樽出しのままの度数で瓶詰め |

## 4. 原料とピート

### 主要原料

| 原料 | 用途 | 風味特性 |
|---|---|---|
| **大麦麦芽（モルト）** | モルトウイスキーの主原料。二条大麦が主 | 穀物の甘さ、ビスケット |
| **トウモロコシ** | バーボン（51%以上）、コーン（80%以上）、グレーン | 甘味、まろやか |
| **ライ麦** | ライウイスキー（51%以上）、カナディアンのフレーバリング | スパイシー、ドライ |
| **小麦** | ウィートウイスキー（51%以上）。メーカーズマークが使用 | ソフト、まろやか |
| **未発芽大麦** | アイリッシュ・シングルポットスチル固有 | クリーミー、スパイシー |

### ピートとフェノール値

**ピート（泥炭）**: 数千年かけて堆積した植物の化石燃料。製麦時の乾燥燃料として使用 → 麦芽にスモーキーな香りを付与。

**フェノール値（PPM: Parts Per Million）** でピート香の強さを表す:

| レベル | PPM | 代表例 |
|---|---|---|
| ノンピート | 0ppm | グレンゴイン |
| ライトリーピーテッド | 1～10ppm | ハイランドパーク（約20ppm） |
| ミディアムピーテッド | 10～25ppm | ボウモア（約25ppm） |
| ヘビリーピーテッド | 25～50ppm | アードベッグ（約55ppm）、ラフロイグ（約40ppm） |
| 超ヘビー | 80ppm以上 | オクトモア（最大309ppm） |

**主なフェノール化合物:**
- **グアイアコール** → スモーキー
- **クレゾール** → タール、石炭
- **オイゲノール** → クローブ

## 5. 蒸留方式

### 単式蒸留器（ポットスチル / Pot Still）

- **構造**: ポット（加熱部）→ スワンネック → ラインアーム → コンデンサー（冷却・液化）
- **素材**: 銅製（銅が硫黄化合物を除去する重要な役割）
- **方式**: 1回の仕込みで1バッチ分（バッチ式）
- **用途**: モルトウイスキーの製造
- スコッチは通常 **2回蒸留**、アイリッシュは伝統的に **3回蒸留**

**ポットスチルの形状と酒質の関係:**

| 形状 | 酒質 | 代表例 |
|---|---|---|
| 背が高い | 軽い | グレンモーレンジィ（スコットランド最長のネック） |
| 背が低い | 重い | マッカラン |
| バルジ型（膨らみ） | 還流↑ → 軽やか | — |
| ラインアーム上向き | 軽い | — |
| ラインアーム下向き | 重い | — |

### 連続式蒸留機（パテントスチル / コラムスチル）

- 背の高い塔の内部が穴の空いたトレイ（棚板）で仕切られた構造
- 連続的に原料を投入・蒸留 → **大量生産**向き
- **用途**: グレーンウイスキーの製造
- 高アルコール度数を得やすく、**クリーンな酒質**

### カフェスチル（コフィースチル）

- **1831年頃**、アイルランド人 **イーニアス・コフィー** が発明・特許取得
- 連続式蒸留機の一種だが、**モロミ塔と清溜塔の2塔のみ**
- 塔が少ない分、原料由来の風味・成分が残りやすい
- 日本では **ニッカウヰスキーの宮城峡蒸溜所** が世界的にも稀少なカフェスチルを保有
  - 銘柄:「ニッカ カフェモルト」「ニッカ カフェグレーン」

## 6. 熟成と樽

### 樽の種類とサイズ

| 樽の種類 | 容量 | 特徴 |
|---|---|---|
| **バレル（Barrel）** | 180～200L | バーボン熟成用。アメリカンオーク |
| **ホグスヘッド（Hogshead）** | 220～250L | バーボン樽を組み直し。スコッチで最も多用 |
| **バット（Butt）** | 約500L | シェリー酒の貯蔵・輸送用。ヨーロピアンオーク |
| **パンチョン（Puncheon）** | 480～520L | シェリー酒貯蔵用 |
| **クォーターカスク（Quarter Cask）** | 約125L | 小型で熟成が速い |

### 樽材の種類

| 樽材 | 風味特性 |
|---|---|
| **アメリカンホワイトオーク** | バニラ、キャラメル、ハチミツの甘い風味 |
| **ヨーロピアンオーク（スパニッシュオーク）** | ドライフルーツ、紅茶、チョコレート、レーズン |
| **ミズナラ（日本固有のオーク）** | 伽羅、白檀のような東洋的な香り |

### ファーストフィルとリフィル

- **ファーストフィル**: 前酒（バーボンやシェリー）を抜いた後、初めてウイスキーを詰める。前酒の影響が強い
- **リフィル**（セカンドフィル以降）: 樽の影響がマイルドに
- 使用回数が増えるほど樽の成分は弱まる

### 熟成樽の由来と風味

| 樽の由来 | 付与される風味 | 備考 |
|---|---|---|
| **バーボン樽** | バニラ、ココナッツ、キャラメル | スコッチの約90%が使用 |
| **シェリー樽** | レーズン、ドライフルーツ、チョコレート、スパイス | マッカランが有名 |
| **ワイン樽** | ベリー系、タンニン | 赤ワイン/白ワイン |
| **ポート樽** | 甘いベリー、プラム | — |
| **ラム樽** | トロピカル、甘さ | — |
| **ミズナラ樽** | 伽羅、白檀 | 日本固有 |

### エンジェルズシェア（天使の分け前 / Angel's Share）

熟成中に樽から蒸発で失われるウイスキーの量。

| 地域 | 年間蒸発率 | 備考 |
|---|---|---|
| **スコットランド** | 約2～3% | 冷涼な気候 |
| **ケンタッキー州** | 初年度10～16%、以降約5% | 寒暖差が激しい |
| **台湾（カバラン等）** | 約17～18% | 熱帯気候 |

- 20年熟成で累計 **約40%** が失われることも
- 樽の寿命は約 **60～70年**

### 熟成年数の影響

- **若い原酒**: アルコールの刺激、穀物感、フレッシュ
- **長期熟成**: 複雑さ、まろやかさ、樽由来の風味が増す
- ラベル表示の年数 = 使用された**最も若い原酒**の熟成年数（スコッチの場合）

## 7. スコッチウイスキー

### 歴史年表

| 年 | 出来事 |
|---|---|
| **1494年** | スコットランド王室財務記録に「アクアヴィテ（生命の水）」初出。修道士ジョン・コーに麦芽8ボルを支給 |
| **1644年** | ウイスキー課税開始 |
| **1707年** | スコットランドとイングランドが合併 → 重税時代へ |
| **1725年** | 麦芽への重税で「密造時代」が本格化 |
| **1823年** | エクサイズ法改正で密造時代終焉 |
| **1824年** | ジョージ・スミスがグレンリベット蒸留所を政府公認第1号として創業 |
| **1826年** | ロバート・スタインが連続式蒸留器を発明 |
| **1831年** | イーニアス・コフィーがカフェ式連続蒸留器を改良・完成 |
| **1853年** | アンドリュー・アッシャーがブレンデッドモルトウイスキーを考案 |
| **1860年** | 法改正でモルトとグレーンのブレンドが正式に認可 → ブレンデッドウイスキー誕生 |
| **1880年代** | フィロキセラ禍でフランスのブランデー・ワインが壊滅 → スコッチの世界的需要が急増 |
| **1909年** | ロイヤル・コミッション裁定（穀物原料の蒸留酒を全てウイスキーと認定） |

### 法的定義（The Scotch Whisky Regulations 2009）

- スコットランドの蒸留所で製造
- 穀類を原料とし、酵母により発酵
- アルコール度数 **94.8%以下** で蒸留
- 容量 **700リットル以下** のオーク樽で **最低3年以上** 熟成
- 最低アルコール度数 **40%** で瓶詰め
- 水と**カラメル色素**以外の添加物不可

## 8. アイリッシュウイスキー

### 歴史年表

| 年 | 出来事 |
|---|---|
| **1172年** | ヘンリー2世のアイルランド侵攻時、「命の水（アクアヴィテ）」の記録 |

### 特徴

- **3回蒸留**が基本（スコッチは通常2回） → クリアでスムーズ

## 10. カナディアンウイスキー

### 製法の特徴

| 構成 | 原料 | 蒸留 | 役割 |
|---|---|---|---|
| **ベースウイスキー** | トウモロコシ主原料 | 連続式蒸留 | ニュートラルで軽い酒質 |
| **フレーバリングウイスキー** | ライ麦主原料 | 単式＋連続式蒸留 | スパイシーでコクのある風味 |

両者をブレンドして製品化。**プレ・ブレンディング**（樽詰め前にブレンド）はカナディアンクラブの特徴。

### 法的定義

- 穀類のみを原料、麦芽のデンプン分解酵素で糖化
- カナダ国内で蒸留・熟成
- 700L以下の樽で **最低3年以上** カナダ国内で熟成
- アルコール度数 **40%以上** で瓶詰め
- **9.09%** までの外国産ウイスキーまたはワイン等のブレンドが許可（独自の規制）

### 歴史

| 年 | 出来事 |
|---|---|
| **1688年** | カナダで初めて蒸留酒が造られる |
| **1858年** | ハイラム・ウォーカーがカナディアンクラブを創業 |
| **1920～1933年** | アメリカ禁酒法時代にカナダからの密輸で繁栄。「アメリカのウイスキー庫」 |
| **1939年** | クラウンローヤル誕生（英国王ジョージ6世のカナダ公式訪問記念） |

### 主要銘柄と蒸留所

| 銘柄 | 蒸留所 | 特徴 |

### ジャパニーズウイスキーの定義（2021年自主基準）

1. **原材料**: 麦芽を必ず使用。穀類と日本国内で採水された水のみ
2. **製造**: 糖化・発酵・蒸留は日本国内の蒸留所で実施
3. **蒸留**: 留出時アルコール度数 **95%未満**
4. **熟成**: 700リットル以下の木製樽に詰め、日本国内で **3年以上** 貯蔵
5. **瓶詰め**: 日本国内で行い、アルコール度数 **40度以上**
6. カラメルの使用は認める

※法律ではなく日本洋酒酒造組合の自主基準（加盟82社が順守）

## 14. テイスティング用語と方法

### テイスティングの手順

1. **外観（色）の確認**: グラスを傾けて光に透かす
   - ペールゴールド → ゴールド → アンバー → カッパー → マホガニー
2. **ノージング（香り）**: グラスをゆっくり回し、鼻を近づける。口を少し開けると繊細な香りが感じられる
3. **テイスティング（味わい）**: 少量を口に含み、**最低10秒**舌の上で転がす
4. **フィニッシュ（余韻）**: 飲み込んだ後の持続する香味。短い/中程度/長いで評価

### 推奨グラス

- **テイスティンググラス（グレンケアングラス）**: チューリップ型、口がすぼまっており香りを集めやすい

### テイスティング表現（フレーバーホイール）

スコッチウイスキー研究所が開発した体系的分類ツール。

| カテゴリ | 代表的表現 |
|---|---|
| **スモーキー/ピーティ** | スモーキー、ピーティー、タール、焦げ、灰、薬品的、ヨード |
| **フローラル** | バラ、ジャスミン、スミレ、ラベンダー、金木犀、ヘザー |
| **フルーティ** | リンゴ、洋梨、柑橘、バナナ、桃、パイナップル、ドライフルーツ、レーズン |
| **スパイシー** | 黒胡椒、シナモン、ナツメグ、クローブ、ジンジャー |
| **スウィート** | ハチミツ、バニラ、キャラメル、トフィー、チョコレート、バタースコッチ |
| **ウッディ/オーキー** | オーク、杉、白檀、松脂、新しい木材 |
| **モルティ/シリアル** | パン、ビスケット、クッキー、トースト、穀物 |
| **ナッティ** | アーモンド、ヘーゼルナッツ、クルミ |

## 16. 近年のトレンド

### クラフトウイスキーブーム

- **日本**: 2025年時点で90超の蒸留所。2年で35か所増加
- **スコットランド**: 10年で50近い新蒸留所がオープン
- **アイルランド**: 20年で蒸留所数が10～15倍に復活

### ワールドウイスキーの台頭

| 国・地域 | 代表的蒸留所 | 特徴 |
|---|---|---|
| **台湾** | カバラン蒸留所（金車グループ） | 亜熱帯気候で急速熟成。国際品評会で多数受賞 |
| **インド** | アムルット蒸留所、ポール・ジョン蒸留所 | 世界最大のウイスキー消費国 |
| **オーストラリア** | サリバンズコーヴ蒸留所（タスマニア） | — |
| **その他** | イスラエル、フランス、スウェーデン、イングランドなど | 各地で蒸留所新設 |

### その他のトレンド

- **原酒不足**: ジャパニーズウイスキーの世界的人気で原酒枯渇。各社増産投資
- **ウイスキー投資/カスク投資**: 希少ウイスキーや樽ごとの投資市場が拡大
- **サステナビリティ**: 再生可能エネルギー、副産物リサイクル
- **ノンエイジ（NAS: No Age Statement）**: 年数表示なしのリリース増加（原酒不足の影響）
- **カスクフィニッシュの多様化**: ワイン樽、ラム樽、テキーラ樽など多様な追加熟成
- **ジャパニーズウイスキーの表示基準制定**（2021年）: 海外産原酒を使った製品との差別化

## 17. 試験対策のポイント

### 3級対策

- 公式テキストを通読し、基礎用語・5大ウイスキーの違いを理解
- スコッチの6産地と主要蒸留所を覚える
- 製造工程の流れを一通り説明できるようにする
- 過去問を繰り返し解く

### 2級対策

- テキストを深く読み込み、各国の法的定義の違いを整理
- 主要蒸留所と銘柄の紐付けを暗記
- ブレンデッドウイスキーのキーモルトを覚える
- 歴史の年号と人物を関連づけて記憶
- 過去問から7～8割出題されるので過去問対策が最重要

### 1級対策

- テキスト+参考図書を網羅的に学習
- 最新の業界動向・時事問題に注目（新蒸留所オープン、受賞歴、法改正等）
- 『Whisky Galore』等の専門誌で最新情報をキャッチアップ
- 各蒸留所の細かい特徴（蒸留器の形状、水源、所有企業等）まで把握
