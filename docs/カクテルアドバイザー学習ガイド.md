# カクテルアドバイザー資格 総合学習ガイド

---

## 1. カクテルアドバイザーとは（概要・認定団体）

### HBAカクテルアドバイザー（主要資格）

- **認定団体**: 一般社団法人 日本ホテルバーメンズ協会（HBA）
- **概要**: カクテルの基礎知識・基礎技術を一般の方（愛好家）向けに認定する民間資格
- **目的**: カクテルの魅力を広く普及させ、飲料文化の発展・育成に貢献
- **対象者**: 20歳以上で、バーや飲食店に勤務していない一般の方（HBA会員は受講不可）
- **費用**: 36,000円（税込、教材・試験料込み）
- **公式サイト**: https://www.hotel-barmen-hba.or.jp/c-adviser

### NBAバーテンダー呼称技能認定試験（プロ向け関連資格）

- **認定団体**: 一般社団法人 日本バーテンダー協会（N.B.A.）
- **概要**: プロのバーテンダーとしての知識・技能を認定する資格
- **種類**:
  - バーテンダー呼称技能認定試験
  - インターナショナルバーテンダー呼称技能認定試験
- **公式サイト**: https://www.bartender.or.jp/qualification/

### 3-1. カクテルの基礎知識（定義・分類・技法）

#### カクテルの定義
- カクテルとは、ベースとなるお酒に他の酒やジュース、シロップなどの副材料を混ぜ合わせた飲料の総称
- 語源には諸説あり（「雄鶏の尾」説、メキシコの少女コクテル説など）

#### カクテルの分類

**スタイルによる分類:**

| 分類 | 特徴 | 容量 | グラス |
|------|------|------|--------|
| ショートドリンク | 氷なし、短時間で飲む、アルコール度数高め | 60〜90ml | カクテルグラス |
| ロングドリンク | 氷入り、ゆっくり飲む、度数低め | 150〜300ml | タンブラー、コリンズグラス等 |

**温度による分類:**
- コールドドリンク（冷たいカクテル）
- ホットドリンク（温かいカクテル）

**アルコール有無:**
- アルコール・カクテル
- ノンアルコール・カクテル（モクテル）

**TPOによる分類:**
- 食前酒（アペリティフ）: さっぱり・辛口系（マティーニ、キールなど）
- 食後酒（ディジェスティフ）: 甘口・濃厚系（アレクサンダー、グラスホッパーなど）
- オールデイ・カクテル: いつでも楽しめるカクテル

### 3-2. ベーススピリッツ（6大スピリッツ）

#### ジン (Gin)
- **原料**: 穀物（大麦・ライ麦・トウモロコシ等）
- **製法**: 蒸留後にジュニパーベリー（杜松の実）等のボタニカルで香り付け
- **主要産地**: イギリス（ロンドン・ドライ・ジン）、オランダ（ジュネヴァ）
- **アルコール度数**: 37.5〜47%程度
- **代表銘柄**: ビーフィーター、タンカレー、ボンベイサファイア、ゴードン、ヘンドリックス
- **代表的カクテル**: ジントニック、マティーニ、ギムレット、ジンフィズ、ネグローニ

#### ウォッカ (Vodka)
- **原料**: 穀物（小麦・ライ麦）、ジャガイモ
- **製法**: 蒸留後に白樺の活性炭でろ過、無色透明・無味無臭が特徴
- **主要産地**: ロシア、ポーランド、スウェーデン、フィンランド
- **アルコール度数**: 40〜50%程度
- **代表銘柄**: スミノフ、アブソルート、グレイグース、ストリチナヤ、ベルヴェデール
- **代表的カクテル**: モスコー・ミュール、ブラッディ・マリー、スクリュードライバー、コスモポリタン、エスプレッソ・マティーニ

#### ラム (Rum)
- **原料**: サトウキビの搾り汁またはモラセス（糖蜜）
- **製法**: 発酵・蒸留後に樽で熟成
- **種類**:
  - ホワイトラム（ライトラム）: 無色透明、軽い味わい
  - ゴールドラム（ミディアムラム）: 淡い琥珀色、中程度の風味
  - ダークラム（ヘビーラム）: 濃い琥珀色、濃厚な風味
- **主要産地**: カリブ海諸国（キューバ、ジャマイカ、プエルトリコ等）
- **アルコール度数**: 40〜50%程度
- **代表銘柄**: バカルディ、ハバナクラブ、マイヤーズ、キャプテンモルガン
- **代表的カクテル**: ダイキリ、モヒート、ピニャ・コラーダ、キューバ・リブレ、マイタイ

#### テキーラ (Tequila)
- **原料**: ブルー・アガベ（リュウゼツラン）
- **製法**: アガベの芯（ピニャ）を蒸して搾汁、発酵・蒸留
- **種類**:
  - ブランコ（シルバー）: 熟成なし、透明
  - レポサド: 2ヶ月〜1年熟成
  - アネホ: 1〜3年熟成
  - エクストラ・アネホ: 3年以上熟成
- **主要産地**: メキシコ（ハリスコ州テキーラ地区を中心に限定地域）
- **アルコール度数**: 35〜55%程度
- **代表銘柄**: ホセ・クエルボ、サウザ、パトロン、ドン・フリオ、エラドゥーラ
- **代表的カクテル**: マルガリータ、テキーラ・サンライズ、パロマ、テキーラ・ショット

#### ウイスキー (Whisky/Whiskey)
- **原料**: 大麦、トウモロコシ、ライ麦、小麦などの穀物
- **製法**: 糖化・発酵・蒸留後に木樽（主にオーク樽）で熟成
- **種類**:
  - スコッチウイスキー: スコットランド産（モルト、グレーン、ブレンデッド）
  - アイリッシュウイスキー: アイルランド産（3回蒸留が伝統）
  - バーボンウイスキー: アメリカ産（トウモロコシ51%以上使用）
  - ライウイスキー: ライ麦51%以上使用
  - カナディアンウイスキー: カナダ産（ライトで滑らか）
  - ジャパニーズウイスキー: 日本産
- **アルコール度数**: 40〜63%程度
- **代表銘柄**: ジョニーウォーカー、バランタイン、ジャックダニエル、メーカーズマーク、山崎、白州
- **代表的カクテル**: マンハッタン、オールド・ファッションド、ウイスキー・サワー、ミント・ジュレップ、ラスティ・ネイル

#### ブランデー (Brandy)
- **原料**: ブドウ（フルーツブランデーの場合はリンゴ、洋ナシなど）
- **製法**: ワインを蒸留して樽で熟成
- **種類**:
  - コニャック: フランス・コニャック地方産（最高級品）
  - アルマニャック: フランス・アルマニャック地方産
  - カルヴァドス: フランス・ノルマンディー地方産（リンゴ原料）
  - グラッパ: イタリア産（ブドウの搾りかす原料）
  - マール: フランス産（ブドウの搾りかす原料）
- **等級**: V.S.（3年以上）、V.S.O.P.（4年以上）、X.O.（6年以上/コニャックは10年以上）
- **アルコール度数**: 40〜43%程度
- **代表銘柄**: ヘネシー、レミーマルタン、クルボアジェ、カミュ、マーテル
- **代表的カクテル**: サイドカー、アレクサンダー、ブランデー・サワー、B&B、ジャック・ローズ

### 3-4. ワイン・ビール・日本酒の基礎知識

#### ワイン
**分類:**
- スティルワイン（非発泡性）: 赤・白・ロゼ
- スパークリングワイン（発泡性）: シャンパーニュ、カヴァ、プロセッコ等
- フォーティファイドワイン（酒精強化）: シェリー、ポート、マデイラ等
- フレーバードワイン（香味付き）: ベルモット（ドライ/スイート）、サングリア等

**主要ブドウ品種:**
- 赤: カベルネ・ソーヴィニヨン、メルロ、ピノ・ノワール、シラー
- 白: シャルドネ、ソーヴィニヨン・ブラン、リースリング

**カクテルで使う重要なワイン:**
- ドライ・ベルモット（フレンチ・ベルモット）: マティーニの副材料
- スイート・ベルモット（イタリアン・ベルモット）: マンハッタンの副材料
- シャンパン: ミモザ、ベリーニ、キール・ロワイヤル

#### ビール
**分類:**
- エール（上面発酵）: ペールエール、IPA、スタウト、ポーター
- ラガー（下面発酵）: ピルスナー、ボック、メルツェン

**カクテルで使うビール:**
- シャンディガフ（ビール＋ジンジャーエール）
- レッドアイ（ビール＋トマトジュース）
- ブラック・ベルベット（スタウト＋シャンパン）

#### 日本酒
**分類:**
| 種類 | 精米歩合 | 醸造アルコール |
|------|----------|--------------|
| 大吟醸 | 50%以下 | あり |
| 純米大吟醸 | 50%以下 | なし |
| 吟醸 | 60%以下 | あり |
| 純米吟醸 | 60%以下 | なし |
| 本醸造 | 70%以下 | あり |
| 純米 | 規定なし | なし |

**特定名称酒と普通酒**の区別、甘口・辛口の違い（日本酒度）なども学習範囲。

### 3-7. カクテルの技法（4大技法 + その他）

#### 1. ビルド (Build)
- **方法**: グラスに直接材料を注ぎ、軽くかき混ぜる
- **特徴**: シェーカーやミキシンググラスを使わない最もシンプルな技法
- **適するカクテル**: 炭酸系（炭酸が抜けにくい）、混ざりやすい材料同士の組み合わせ
- **代表例**: ジントニック、スクリュードライバー、カンパリソーダ

#### 2. ステア (Stir)
- **方法**: ミキシンググラスに氷と材料を入れ、バースプーンで静かにかき混ぜ、ストレーナーで濾してグラスに注ぐ
- **特徴**: 混ざりやすい酒同士を合わせる。シェイクより氷が溶けにくく、繊細な味わいを保つ。透明感のある仕上がり
- **適するカクテル**: アルコールのみの組み合わせ、澄んだカクテル
- **代表例**: マティーニ、マンハッタン、ネグローニ

#### 3. シェイク (Shake)
- **方法**: シェーカーに氷と材料を入れ、リズミカルに振る
- **特徴**: 短時間で材料を均一に混ぜ合わせ、急速に冷却。空気を含ませてまろやかにする。混ざりにくい材料（ジュース、クリーム、卵等）の混合に適する
- **ポイント**: 振りすぎると水っぽくなる。10〜15秒程度が目安
- **代表例**: ダイキリ、サイドカー、コスモポリタン

#### 4. ブレンド (Blend)
- **方法**: ブレンダー（ミキサー）に材料と氷を入れて電動で撹拌
- **特徴**: フローズンスタイルのカクテルやフルーツを使ったカクテルに最適。シャーベット状の仕上がり
- **代表例**: フローズン・ダイキリ、フローズン・マルガリータ、ピニャ・コラーダ

#### その他の技法
| 技法 | 説明 |
|------|------|
| フロート (Float) | 比重の違いを利用して液体を層にする（プース・カフェスタイル） |
| マドル (Muddle) | フルーツやハーブをペストルで潰して香りや風味を出す |
| レイヤー | 複数の液体を比重差で層状に重ねる |
| リンス | グラスの内側にリキュールやスピリッツを薄く塗る（サゼラックのアブサンリンスなど） |

### 3-11. 酒類の歴史

#### 酒の起源と発展
- **醸造酒の起源**: 紀元前数千年前、メソポタミア・エジプトでビール・ワインの醸造が始まる
- **蒸留技術の発展**: 8〜12世紀頃、アラビアの錬金術師が蒸留技術を発展（「アルコール」の語源はアラビア語「al-kuhl」）
- **リキュールの誕生**: 中世ヨーロッパの修道院で薬草酒として発展

#### 各スピリッツの歴史
- **ジン**: 17世紀にオランダの医師シルヴィウスが薬用酒として開発。18世紀イギリスで「ジン狂時代」
- **ウォッカ**: 12世紀頃にロシアまたはポーランドで誕生
- **ラム**: 17世紀カリブ海でサトウキビのプランテーションと共に発展
- **テキーラ**: 16世紀スペイン征服者がメキシコでアガベの蒸留酒を製造
- **ウイスキー**: 15世紀にアイルランドまたはスコットランドで修道士が蒸留。「生命の水（ウシュクベーハ）」が語源
- **ブランデー**: 16世紀にオランダ商人がワインを蒸留。「brandewijn（焼いたワイン）」が語源

#### カクテルの歴史
- 19世紀前半にアメリカで「カクテル」という言葉が登場（1806年に新聞で初めて定義）
- 19世紀後半の「カクテルの黄金時代」（Jerry Thomas著『Bar-Tender's Guide』1862年）
- 1920-1933年のアメリカ禁酒法時代 → 多くのバーテンダーがヨーロッパ・カリブ海に移住しカクテル文化が世界に拡散
- 戦後〜現代のカクテル・ルネサンス

## 7. 重要用語集

| 用語 | 意味 |
|------|------|
| アペリティフ (Aperitif) | 食前酒 |
| ディジェスティフ (Digestif) | 食後酒 |
| オン・ザ・ロック (On the Rocks) | 氷を入れたグラスで飲むスタイル |
| ストレート (Straight/Neat) | 氷なし・水なしで飲むスタイル |
| アップ (Up) | シェイクまたはステア後に氷なしで提供 |
| ダッシュ (Dash) | ビターズ等の少量（約1ml） |
| ティースプーン (tsp) | 約5ml |
| フロート (Float) | 液体を浮かせる技法 |
| スノースタイル | グラスの縁に塩や砂糖を付けるスタイル |
| プース・カフェ (Pousse Cafe) | 比重の異なる液体を層にしたカクテル |
| チェイサー (Chaser) | 強い酒の後に飲む水やビール |
| ミクソロジー (Mixology) | カクテル調合の技術・学問 |
| バック (Buck) | スピリッツ＋レモンジュース＋ジンジャーエール |
| フィズ (Fizz) | スピリッツ＋柑橘ジュース＋砂糖＋ソーダ |
| サワー (Sour) | スピリッツ＋柑橘ジュース＋砂糖 |
| コブラー (Cobbler) | クラッシュドアイス＋酒＋フルーツ |
| スリング (Sling) | スピリッツ＋レモン＋砂糖＋水/ソーダ |
| コリンズ (Collins) | スピリッツ＋レモン＋砂糖＋ソーダ（長いグラス） |
| リカール (Ricard) | アニス風味のフランス産リキュール |
| ビターズ (Bitters) | 苦味の強い香味料（アンゴスチュラ、ペイショーズ等） |
| プルーフ (Proof) | アルコール度数の表示法（米国: Proof = 度数x2） |
| ABV | Alcohol By Volume の略。アルコール度数のこと |
| IBA | International Bartenders Association（国際バーテンダー協会） |
| NBA | Nippon Bartenders Association（日本バーテンダー協会） |
| HBA | Hotel Barmen's Association of Japan（日本ホテルバーメンズ協会） |
