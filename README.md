# 🍸 カクテルマスター

**カクテルアドバイザー資格試験対策 学習Webアプリ**

HBA（日本ホテルバーメンズ協会）認定カクテルアドバイザー資格の学習を、楽しくビジュアルに行えるWebアプリケーションです。

## 機能

| 機能 | 説明 |
|------|------|
| 📖 図鑑 | スピリッツ・リキュール・カクテルレシピをカード型UIで閲覧 |
| ❓ クイズ | カテゴリ別4択クイズ、正解率トラッキング |
| 🃏 暗記カード | スワイプ式フラッシュカード（用語・レシピ・ツール） |
| 🌳 飲料分類 | インタラクティブな飲料分類ツリー |
| 🔧 バーツール | バーツール・グラスの図鑑 |

## セットアップ

```bash
npm install
npm run dev
```

## 技術スタック

- React 19 + TypeScript
- Vite 8
- Tailwind CSS v4
- Framer Motion
- React Router v7

## 学習コンテンツ

- 6大ベーススピリッツ（ジン、ウォッカ、ラム、テキーラ、ウイスキー、ブランデー）
- リキュール30種以上
- カクテルレシピ60種
- バーツール18種、グラス14種
- 4大調製技法
- IBA公認カクテル101種リスト
- 酒類の歴史・飲料分類体系
